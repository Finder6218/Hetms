# 高等教育培训管理系统

软件主要适用于针对高等教育进行培训的机构管理学员、师资等方面的信息。

系统包含： 

学员档案：学员档案   
师资档案：师资档案   
培养计划：课程信息   
查询汇总：已毕业学员查询、已进入论文答辩学员、欠费学员汇总、学员考试成绩查询、所学课程查询、学员考勤、学员领书、学员论文查询功能。   

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
